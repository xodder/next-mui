# next-mui
A NextJs starter kit for [mui](https://mui.com/material-ui).

## Getting Started
To create a new project using this as a template, run:

```console
$ pnpm create next-app --example https://github.com/xodder/next-mui <project-name>
```

## Directory Structure
```markdown
<project>
├── public
├── scripts
└── src

    ├── components
    │   ├── generics
    │   ├── impls
    │   ├── layouts
    │   └── shared
    ├── config
    ├── guards
    ├── helpers
    ├── pages
    ├── theme
    └── utils
```
